How To Apply?

Apply online or complete an official application form, which can be obtained from the Admissions Office, upon payment of a non-refundable application fee. No application is valid unless the appropriate application fee has been paid. Application forms can be downloaded from the University website (Get Application Forms). Application fees can be deposited into one of the following University Account: ZB Bank Account Number: 4565 386395 200
